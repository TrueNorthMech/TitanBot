# TitanBot: 4-Wheeled Holonomic Robot Platform

![TitanBot Header](https://img.shields.io/badge/Project-TitanBot-blue)
![License](https://img.shields.io/badge/License-MIT-green)
![CAD](https://img.shields.io/badge/CAD-FreeCAD_1.0.2-orange)

TitanBot is a professional-grade, open-source robotics platform inspired by the rugged aesthetic of the Humvee. It features a modular "T-Profile" chassis architecture, integrated cable management, and a holonomic drive system for omnidirectional movement.

- **/CAD**: Master FreeCAD (.FCStd) files for all structural components.
- **/STLs**: Ready-to-print mesh files for 3D printing.
- **/Firmware**: Arduino sketches for kinematic control and sensor integration.
- **/Docs**: Wiring diagrams, Bill of Materials (BOM), and assembly guides.
